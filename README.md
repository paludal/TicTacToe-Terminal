# TicTacToe-Terminal

This project(Tic-Tac-Toe Game) is about client-server comunication using socket on terminal in nodejs

Steps to run the project

1) clone the project from https://github.com/paludal/TicTacToe-Terminal.git
2) after clone run npm install which automatically install all dependencies of the project
3) run node server.js in one terminal to run server
4) run node client.js in two different terminals after that only start the game
